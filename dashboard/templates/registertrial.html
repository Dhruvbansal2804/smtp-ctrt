Interventions

For each arm of the trial record a brief intervention name plus an intervention description.

If the intervention is based on one or more drugs, please use the International Nonproprietary Name for each drug if possible, instead of brand/trade names. For unregistered drugs, the acceptable options are the generic name, chemical name, or company serial number.

If the intervention consists of several separate treatments, please list them all in one line separated by commas. For example: "low-fat diet, exercise".

In controlled trials, it is important to clearly identify the control arm. The control intervention(s) refers to the interventions against which the study intervention is evaluated. Examples include placebo, no treatment, or active control. If an active control is used, please enter the name(s) of the intervention. Otherwise, use "placebo" or "no treatment" as applicable. Additionally, provide other relevant intervention details such as dose, duration, mode of administration, etc.

{{ form.intervention_name.label|title }}

  • For investigational new drugs without a generic name, temporary use of a chemical name, company code, or serial number is acceptable. Update the associated registered records once the generic name is established.
  • For non-drug interventions, provide a detailed intervention name that distinguishes it from similar interventions.
